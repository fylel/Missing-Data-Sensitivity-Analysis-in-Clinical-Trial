題目： 臨床試驗缺失資料敏感度分析
本專案旨在探討臨床試驗資料中缺失值（Missing Data）的處理與其對研究結論的影響。於臨床研究中，缺失值為常見現象，可能源自資料輸入錯誤、受試者退出試驗、非人為疏失或資料蒐集困難等因素。若未適當處理，缺失值可能導致研究結果產生偏差、降低統計檢定力，甚至產生錯誤結論。雖然多數統計軟體預設以刪除含缺失值樣本（Complete Case Analysis）作為處理方式，但此方法可能同時刪除其他具價值的觀測資料，進而影響推論結果。因此，本研究以一份重度憂鬱症（Major Depressive Disorder, MDD）臨床試驗資料為例，系統性地進行缺失資料探索性分析，並比較多種缺失資料處理方法，包括完整個案分析、單一插補、多重插補、Reference-Based Imputation 以及 Tipping Point Analysis，透過敏感度分析評估不同方法下研究結論的不確定性與穩健性，建立完整且嚴謹的臨床試驗缺失資料分析流程。
